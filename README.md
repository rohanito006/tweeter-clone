Description

Clone simplifié de Twitter développé avec AdonisJS et React, permettant aux utilisateurs de publier du contenu, interagir et gérer leurs relations sociales.

  Fonctionnalités
Authentification sécurisée (login/register)
Publication de posts avec médias
Système de likes et commentaires
Système d’abonnements (followers/following)
API REST structurée
  Technologies
Backend : AdonisJS, Node.js, TypeScript
Frontend : React
Base de données : PostgreSQ
